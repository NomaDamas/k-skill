# 다모다란식 한국 상장사 브리지 가이드

## 이 기능으로 할 수 있는 일

- 한국 상장사를 Aswath Damodaran 식으로 해석
- 회사 스토리를 성장 / 마진 / 재투자 / 리스크로 번역
- 현재 가격이 요구하는 기대를 말로 풀어 설명
- `이 가격이 성립하려면`과 `깨지는 조건`을 분리해서 정리

## 언제 특히 유용한가

- 플랫폼/바이오/AI/정책 테마처럼 서사가 강한 종목
- 턴어라운드 기대가 선반영된 종목
- 좋은 회사인지와 좋은 가격인지를 분리해 보고 싶은 경우

## 추천 분석 순서

1. `korean-listed-company-analysis`로 기본 팩트와 기대 구조를 먼저 잡는다.
2. 회사의 핵심 스토리를 한 문장으로 정의한다.
3. 그 스토리를 성장, 마진, 재투자, 리스크 4축으로 내린다.
4. 현재 시총/주가가 요구하는 기대를 적는다.
5. `이 가격이 유지되려면` 조건과 `깨지는 조건`을 따로 적는다.
6. 마지막에 `스토리는 좋지만 가격도 이미 반영 / 숫자가 받쳐줌 / 확인 더 필요 / 기대 과함` 중 하나로 정리한다.

## 답변 템플릿 권장

- 총평
- 지금 시장이 믿는 스토리
- 그 스토리가 요구하는 숫자
- 현재 가격이 깔고 있는 기대
- 이 가격이 성립하려면
- 반대로 깨지는 조건
- 결론

## 해석 팁

- 한국 시장은 수급과 테마가 서사를 과장할 수 있으니 공시와 실적으로 다시 접지한다.
- 좋은 회사와 좋은 주식은 다를 수 있다는 문장을 피하지 않는다.
- 데이터가 부족하면 valuation 숫자를 억지로 확정하지 말고 기대치 비교 중심으로 쓴다.
- 뉴스 헤드라인만으로 스토리를 강화하지 않는다.

## 에러/제약

- 공식 공시가 비어 있으면 해석 자신감을 낮춘다.
- 적자 성장주는 마진보다 생존과 재투자 부담을 더 본다.
- 지주사/복합기업은 단일 서사로 과단순화하지 않는다.
